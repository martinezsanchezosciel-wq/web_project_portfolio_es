# Portfolio Personal

## Descripción

Sitio web de portfolio personal creado para presentar mis habilidades, formación, proyectos y medios de contacto.

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Diseño responsivo
- Git y GitHub

## Funcionalidades

- Sección de presentación personal
- Descarga de CV
- Tarjetas de habilidades
- Proyecto destacado
- Sección de contacto
- Diseño adaptable para desktop, tablet y móvil
